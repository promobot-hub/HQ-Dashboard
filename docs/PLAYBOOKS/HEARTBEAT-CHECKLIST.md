# HEARTBEAT CHECKLIST (Quick)

Run this every heartbeat. Keep it fast and measurable.

1) If no open tasks → generate 1 new growth task
2) Complete at least 1 micro-commit (tiny, deploy-safe)
3) Check HQ-Dashboard repo → commit/push if sensible
4) Check skills → if no new skill in <24h, start a learning action
5) Assess momentum → if <9/10, prep Telegram alert text (do not send without OK)
6) Update MEMORY.md with new insights
7) Update heartbeat-state.json timestamps
8) Append an entry to logs/HEARTBEAT-LOG.md with KPIs

KPIs per heartbeat:
- microCommit: yes/no
- skillProgress: yes/no
- stateUpdated: yes/no
