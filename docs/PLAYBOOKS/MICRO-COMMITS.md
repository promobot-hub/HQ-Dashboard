# MICRO-COMMITS (Ready-to-Ship)

Ultra-small, deploy-safe tasks you can do in <5 min. Do one every run.

- Create/refresh logs/HEARTBEAT-LOG.md entry with KPIs
- Update heartbeat-state.json timestamps and KPIs
- Add a new checklist/playbook under docs/PLAYBOOKS/
- Improve docs: clarify one line in SOUL.md or TOOLS.md
- Create tasks/TODO-<date>.md with 3 laser-focused items
- Add a stub page/component to HQ-Dashboard (no external calls)
- Add a learning note under docs/LEARNINGS/<date>.md (1-3 bullets)
- Tighten wording in USER.md (typo/clarity pass)
- Add a script stub under scripts/ (no side effects)
- Add a test placeholder file to keep coverage discipline
